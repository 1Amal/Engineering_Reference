                <li>
                  <code>sudo reboot</code>
                  : This command will restart the Raspberry Pi immediately.
                </li>
                <li>
                  <code>shutdown -h now</code>
                  : This is to stop the Raspberry Pi immediately. You can
                  replace “now” by a specific time (shutdown -h 12:05). Don’t
                  use the power switch to stop your Raspberry, you should do it
                  properly by using this command.
                </li>
                <li>
                  <code>shutdown -h 0 </code>: Shutdown computer immediately
                </li>

                <li>
                  <code>ps aux</code>
                  : This command displays all running process on your Raspberry
                  Pi.
                </li>

                 <li>
                  <code>`kill 12345` `kill [pid]`</code>
                  : The kill command allows you to terminate a process. You’ll
                  need the process ID to do this.
                </li>
                <li>
                  <code>`kill -9 12345`</code>
                  :Sometime you may need to use the -9 option to force all
                  related commands to stop. For example, if you run 20 commands
                  in a script and kill it, it’ll continue to the next line, not
                  exit the program, except if you use the -9 option.
                </li>
                <li>
                  <code>`killall php`</code>
                  : Killall to stop all occurrences of a program.
                </li>
